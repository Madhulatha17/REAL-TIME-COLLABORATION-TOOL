# REAL-TIME-COLLABORATION-TOOL

## Project Overview

A MERN Stack Real-Time Collaborative Whiteboard that allows multiple users to draw simultaneously.

## Features

- Real-Time Synchronization
- Multi-User Support
- WebSocket Communication
- Collaborative Whiteboard
- Clear Board Functionality

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- Socket.IO

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Ports

Backend: 5000

Frontend: 3000
